= file_hash_check

The file_hash_check is Created By Lazarus 1.2.4 (http://www.lazarus.freepascal.org/ ).

== Requirement

FPC 2.4.x or later and Rake (any recent version) (http://sourceforge.net/projects/freepascal/ )

--dcpcrypt-2.0.4.1 component (http://www.cityinthesky.co.uk/opensource/dcpcrypt/)--

dcpcrypt-2.0.4.1 component (https://github.com/StephenGenusa/DCPCrypt)

== Features include
 - File Drop check
 - md5 hash print
 - sha1 hash print
 - sha256 hash print
 - VT (VirusTotal.com) Search using Browser
